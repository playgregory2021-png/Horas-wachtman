# Horas Wachtman ⏰

App Android para rastrear horas trabajadas con calendario colorido y estadísticas anuales.

## Características 🎨
- 📅 Calendario con todos los meses del año
- ⏰ Registro de horas trabajadas por día
- 💾 Almacenamiento local (Sin necesidad de internet)
- 📊 Suma automática de horas por mes
- 📈 Estadísticas anuales detalladas
- 🎨 Diseño colorido y moderno
- 🎭 Ícono personalizado

## Tecnologías 🛠️
- Kotlin
- Android SDK
- Jetpack Compose (UI moderna)
- Room Database (Almacenamiento local)
- Material Design 3

## Instalación
1. Clona el repositorio
2. Abre en Android Studio
3. Selecciona un dispositivo o emulador
4. Ejecuta la app
